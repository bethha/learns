Workflows

Command

  • cd change directory
  • mkdir make directory
  • pwd find out which directory you are at
  • cd .. go to one directory on top
  • ls or ls -l list contents (-l) with more details
  • touch 'file.extension' creates a file within a directory

Git Commands

  • git init initialize git process
  • git status check to see the status of files in a repo
  • git commit -m 'reason for comiting'
  • git checkout -- .
  • git add --all add all files in a repo to be committed
  • git remote add origin https://githib.com/location/file.git adding the location to where the file has to be pushed into the server
  • git commit -a -m 'comment' comminting the files
  • git remote - v checking the location to push at the moment
  • git push origin master pushing the files into the server

NPM

You can use NPM as a 'shopping' for code packages.

You do it through the terminal. To install a jquery and a normalize.css package you need to insert the following commands:

  • npm init to start npm
  • npm install jquery or replace jquery by any other package of choice. This will only install the package, but ideally, do as follows:
  • install jquery --save-- save will save the information that you installed jquery in your repo adding this information to the package.jsonfile, under 'dependencies'. There you will also see the version of the package(s) which was installed. This file will save all the information on all the packages which were installed in a repo. After each installation, always add --save to keep this file up to date.
  • Also, in case the packages have been deleted, you can also install everything again if you only have the package.json file. For that, inside the repo type npm install. That will install all directories with the packages files again, which were lost.
  • After installing all the needed packages, rememeber to push them to the server. Just make sure to add the package.json file to the repo, add it to the staging area, commit it and push it to my github account.
  • if you want to check the version of the package which you have installed, just type npm jquery -v. Or replace 'jquery' by the desired package
  • In order to know which package to install look in npm website or google the action a package should perform followed by 'npm'
